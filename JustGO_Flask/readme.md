# 문화데이터 활용 경진대회를 위한 서버
## 개요
- 각 서버별 역할을 분담하여 Node.js, Flask로 분업화
- 데이터베이스로는 MongoDB를 사용
- 두 서버를 호스팅하여 기능별로 구분하거나 Redirect를 이용할 것으로 예상됨

## 파일 구성
    └ static
    └ templates
    └ __init__.py
